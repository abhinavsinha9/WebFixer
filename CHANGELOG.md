# Changelog

All notable changes to the BugFinder platform will be documented in this file.

## [1.0.0] - Initial Release

### Added
- **Authentication System:** Complete JWT-based auth with signup, login, profile updates, and password reset flows.
- **Frontend Architecture:** React 18, Vite, Tailwind CSS with a custom dark-mode glassmorphism design system.
- **Landing Page:** Premium SaaS landing page with animated hero, features, workflow, pricing, and FAQ.
- **Dashboard:** Interactive statistics, health score visualizers, and recent project tracking.
- **Project Import:** Support for uploading project ZIP files.
- **Analysis Engine:** Custom Node.js service that statically analyzes HTML, CSS, and JS files for bugs, security flaws, performance bottlenecks, and accessibility issues.
- **AI Integration:** OpenAI GPT-4o-mini integration for intelligent code review and refactoring suggestions, with a robust rule-based fallback.
- **Bug Tracker:** Comprehensive bug listing, filtering by severity/status, and detailed bug view with code snippets and commenting.
- **Code Explorer:** Interactive file tree and integrated Monaco Editor for viewing source code.
- **Report Generation:** Automated generation of PDF, CSV, and Excel reports using PDFKit and ExcelJS.
- **Admin Panel:** Role-based dashboard for managing users and viewing platform statistics.
- **Settings & Profile:** Preferences management and account controls.
